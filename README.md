# Nitro Technical Test
Nitro Exercise for Frontend Engineer


## Tech Stack

**Client:** HTML, SASS, Javascript

**Server:** Node v18.3.0, Express


## How run project

To install dependencies run:
```bash
  npm install
```

To run project:
```bash
  npm run start
```

or

```bash
  node index.js
```
