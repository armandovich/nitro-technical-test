# Nitro Technical Test
Nitro Exercise for Frontend Engineer


## Tech Stack

**Client:** HTML, SASS, JavaScript

**Server:** Node v18.3.0, Express


## Steps To Run Project

To install dependencies run:
```bash
  npm install
```

To run project:
```bash
  npm run start
```

or

```bash
  node index.js
```

Open localhost on prefered browser:
```bash
http://localhost:3000
```
